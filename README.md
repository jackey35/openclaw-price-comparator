# 云平台 OpenClaw 部署成本对比

直观的比价 Web 应用，对比各家云厂商一键部署 OpenClaw 的成本。

## 功能特性

- 🏷️ 支持多家云厂商对比
- 💰 实时/静态价格展示
- 📊 可视化成本分析
- 🔄 定期更新定价信息

## 支持的云厂商

- 阿里云
- 腾讯云
- 华为云
- AWS
- GCP
- Azure
- 火山引擎
- 百度智能云

## 部署

```bash
# 本地预览
cd cloud-price-comparator
python3 -m http.server 8080
```

然后访问 http://localhost:8080
